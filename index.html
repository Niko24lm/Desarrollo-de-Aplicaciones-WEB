Quiénes Somos

Este proyecto académico tiene como finalidad demostrar el uso de HTML5, CSS3 y Bootstrap para la creación de sitios web modernos. A través de este trabajo se aplican conocimientos relacionados con la estructura semántica, diseño responsivo, componentes visuales y publicación de proyectos web utilizando GitHub y GitHub Pages. Además, permite desarrollar habilidades prácticas para la construcción de interfaces atractivas y funcionales adaptadas a diferentes dispositivos.
