Quiénes Somos

Este proyecto ha sido desarrollado con fines académicos dentro de la asignatura Desarrollo de Aplicaciones Web. Su propósito es aplicar los conocimientos básicos de HTML5 para crear sitios web organizados y funcionales. A través de este trabajo, se busca comprender la importancia de la estructura semántica, el uso adecuado de etiquetas y la integración de diferentes elementos multimedia. Además, este proyecto permite fortalecer habilidades relacionadas con el diseño de páginas web y la gestión de versiones mediante GitHub. El aprendizaje obtenido servirá como base para el desarrollo de aplicaciones web más complejas en futuras unidades de estudio.

Servicios o Productos

  • Diseño básico de páginas web.
  • Organización de contenido mediante HTML5.
  • Creación de menús de navegación.
  • Integración de imágenes y videos.
  • Publicación de proyectos en GitHub.
